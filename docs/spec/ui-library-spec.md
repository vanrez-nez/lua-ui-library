# UI Library Specification

This specification is split into two authoritative documents:

- [UI Foundation Specification](/Users/vanrez/Documents/game-dev/lua-ui-library/docs/spec/ui-foundation-spec.md)
- [UI Controls Specification](/Users/vanrez/Documents/game-dev/lua-ui-library/docs/spec/ui-controls-spec.md)

The foundation document defines the shared architecture, runtime model, propagation, focus, responsive layout, render-skin contract, and theming contract.

The controls document defines the concrete control families built on that foundation.

This file remains the stable entry point for the specification set.
