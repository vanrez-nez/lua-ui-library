# Project Memory

- All code in this project lives under the authority of a specification document.
- If the code does not follow the spec, do not dodge the responsibility to bring that up and proceed to create a discussion on how to resolve the disconnect.
- Never touch the spec contents unless the user specifically instructs you to do so.
- Spec patches must go through a patch document to be discussed.
- If you are making a demo or screen, it is mandatory to read `demos/README.md` first.
